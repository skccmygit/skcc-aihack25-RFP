스포츠 e-러닝 포털 요구사항 명세서

1. 프로젝트 개요

  • 체육인재개발원 학습운영시스템 구축
  • 교육과정 개설, 신청, 수강, 수료증 발급 등의 기능 포함
  • 관리자, 학습자, 강사 등 다양한 사용자 역할 고려

2. 기능 요구사항

REQ-001001 - 웹/모바일 기반 교육플랫폼 개발

  • 분류: 공통 사용자 기능
  • 중요도: 상
  • 근거: RFP_16p_SFR-001
  • 설명:
    • 표준화된 프레임워크(전자정부 표준프레임워크 포함)를 기반으로 개발
    • 웹접근성, 웹호환성, 소프트웨어 개발보안을 철저히 준수하여 개발
    • 웹브라우저 호환성 확보를 위해 크로스 브라우징 지원(Chrome, Edge, FireFox, Safari 등)
    • 웹사이트 내 개인정보는 최소로 활용하고 반드시 암호화
    • 인터넷 웹페이지는 모바일 기기에서도 이용 가능하도록 반응형 웹으로 구축

REQ-001002 - 교육 정보시스템 정보구조 및 UI/UX

  • 분류: 사용자 인터페이스
  • 중요도: 상
  • 근거: RFP_17p_SFR-002
  • 설명:
    • 사용자 친화적인 정보구조 설계
    • 사용자 경험과 편의성을 보장하는 UI/UX 구성
    • 웹 표준 및 웹 호환성을 고려한 반응형 웹페이지 구성
    • 모바일 환경에서 교육 콘텐츠 활용이 가능하도록 구성
    • 교육운영 과정에 대한 전체 목록 기능 제공
    • 회원별 상태에 맞는 추천 교육과정 정보 제공
